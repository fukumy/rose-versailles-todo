# La Pensée de Versailles 🌸
~ 可憐なる日々の備忘録 ~

「ベルサイユの薔薇」の優雅さをそのままに、パンジーをモチーフとした、可憐で使いやすいTODOリストアプリです。

## 🌹 特徴
- **エレガントなデザイン**: 紫と黄色を基調とした、ロココ調の華やかなインターフェース。
- **可憐な演出**: タスクを追加するとパンジーが咲き、完了すると輝くアニメーション。
- **シンプル操作**: 誰でも簡単に「想い（タスク）」を管理できます。

## 🌐 ライブデモ
こちらから実際にアプリを体験いただけます：
[https://fukumy.github.io/rose-versailles-todo/](https://fukumy.github.io/rose-versailles-todo/)

## 🛠️ 技術スタック
- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)

## 📖 使い方
1. **追加**: 入力欄に内容を書き込み、🌸ボタンを押してください。
2. **完了**: タスク横の🌸をクリックすると✨に変わり、完了状態になります。
3. **削除**: 「×」ボタンでタスクを記憶から消去します。
